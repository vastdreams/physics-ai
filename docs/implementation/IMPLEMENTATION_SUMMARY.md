## API Endpoints Summary

### Core Endpoints
- `/api/v1/simulate` - Physics simulations
- `/api/v1/nodes/*` - Nodal graph operations
- `/api/v1/rules/*` - Rule management
- `/api/v1/evolution/*` - Code evolution
- `/api/v1/cot/*` - Chain-of-thought logs

### VECTOR Endpoints
- `/api/v1/vector/delta-factors` - Add delta factors
- `/api/v1/vector/throttle` - Throttle variance
- `/api/v1/vector/bayesian-update` - Bayesian updates
- `/api/v1/vector/overlay-validation` - Model validation
- `/api/v1/vector/statistics` - VECTOR statistics
